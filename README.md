# ML с нуля — интерактивный курс про веса и обучение модели

Бесплатное интерактивное приложение для новичков: разберитесь, что такое признаки,
веса и метки, как модель учится на ошибках и почему данные решают всё.
**10 модулей с живыми песочницами на JavaScript.**

> Никакой магии — только числа, которые меняются.

## Возможности

- **10 интерактивных модулей** с живыми песочницами на React + TypeScript
- **Прогресс сохраняется** локально в `localStorage` — ответы не уходят на сервер
- **Светлая/тёмная тема** с переключателем в углу и плавной инверсией цветов
- **Кнопка-мозг «наверх»** с пульсирующей анимацией
- **32 курируемых бесплатных ресурса** для дальнейшего изучения
- **Адаптивный дизайн** — работает на мобильных и десктопах
- **Доступность**: keyboard-friendly, `aria-label`, `prefers-reduced-motion`

## Модули

1. Термины: признаки, веса, метки
2. Веса: как модель «взвешивает» признаки
3. Ошибка: как модель понимает, что ошиблась
4. Обучение: градиентный спуск без формул
5. Данные: почему данные решают всё
6. Будущее: переобучение и регуляризация
7. Цикл обучения: итерации и сходимость
8. Спам-фильтр: практический пример
9. Объяснимость: почему модель так решила
10. Что дальше: roadmap для развития

## Технологии

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **TypeScript**
- **Tailwind CSS 4** + **shadcn/ui**
- **lucide-react** для иконок
- **Prisma** (для будущих расширений)

## Локальный запуск

```bash
# Установка зависимостей
bun install

# Разработка
bun run dev

# Production-сборка
bun run build

# Запуск production-сервера
bun run start

# Линт
bun run lint
```

Приложение будет доступно на `http://localhost:3000`.

## Структура проекта

```

├── app/
│   ├── layout.tsx          # Корневой layout: тема, scroll-top, футер

    └── use-progress.ts          # Хук прогресса (localStorage)
```
